## 5. 주요 UI 패턴

### 버튼 스타일
```css
/* Primary Button */
background: #00A77C;
color: #FFFFFF;
border-radius: 4px;
padding: 12px 24px;

/* Secondary Button */
background: #505BF0;
color: #FFFFFF;
```

### 상품 카드
```
┌─────────────────────────┐
│ [할인율 뱃지: 100%_]    │
│                         │
│     [상품 이미지]        │
│                         │
│ 상품명 (프로모션 태그)    │
│ ─────────────────────── │
│ 출고가: ₩1,200,000      │
│ 최종가: ₩0 (강조)        │
│ ─────────────────────── │
│ 통신사 | 가입유형 | 할인  │
│ ─────────────────────── │
│ 끝장폰추가할인: -₩xxx    │
│ 공통지원할인: -₩xxx      │
│                         │
│    [신청하기 버튼]       │
└─────────────────────────┘
```

**카드 특징:**
- 할인율 뱃지 (좌상단)
- 가격 계층 구조 (출고가 → 최종가)
- 할인 내역 상세 표시
- 통신사/가입유형 정보

### 캐러셀/슬라이더
- **라이브러리**: Swiper.js
- **사용 위치**: 베스트딜, 핫딜, 브랜드별 섹션, 리뷰

### 필터 사이드바 (목록 페이지)
```
┌─────────────────┐
│ 제조사          │
│ □ 삼성  □ 애플  │
│ □ LG   □ 기타  │
├─────────────────┤
│ 통신사          │
│ ○ SKT ○ KT    │
│ ○ LG U+       │
├─────────────────┤
│ 가입유형        │
│ □ 신규 □ 번이  │
│ □ 기변         │
├─────────────────┤
│ 가격대          │
│ [범위 선택]     │
└─────────────────┘
```

### 탭 네비게이션
- 활성 탭: `#00A77C` 배경 또는 하단 보더
- 비활성 탭: 투명 배경, 회색 텍스트

### 게시판 리스트
- 테이블 형식
- 카테고리 탭 필터
- AJAX 기반 동적 로딩
- 페이지네이션

## 6. 페이지 그룹핑 (템플릿별)

**특징:**
- 풀 너비 히어로 배너
- 다중 캐러셀 섹션 (베스트딜, 핫딜, 브랜드별)
- 리뷰 섹션
- 프로모션 배너

---

### Template B: 상품 목록 페이지
**사용 페이지:**
- `/phone/list/5` (베스트)
- `/phone/list/16` (키즈폰기획전)
- `/phone/list/6` (핫딜)
- `/phone/list/2` (Galaxy)
- `/phone/list/3` (iPhone)

**특징:**
- 좌측 필터 사이드바
- 우측 상품 그리드 (캐러셀/리스트)
- 필터 AJAX 동적 로딩
- "더보기" 버튼

---

### Template C: 상품 상세 페이지
**사용 페이지:**
- `/phone/detail/[ID]`

**특징:**
- 상품 이미지 갤러리
- 옵션 선택기 (통신사, 요금제, 색상)
- 가격 계산기
- 할인 내역 테이블
- 신청 버튼 (CTA)
- 관련 상품 추천

---

### Template D: 게시판 목록 페이지
**사용 페이지:**
- `/board/list/qna` (고객센터)

- `/board/list/faq` (FAQ)

**특징:**
- 카테고리 탭
- 테이블 형식 리스트
- 검색 기능
- 페이지네이션
- 글쓰기 버튼

---

### Template E: 게시판 상세 페이지
**사용 페이지:**
- `/board/detail/plan/[ID]`

**특징:**
- 제목 + 날짜 헤더

- 목록으로 버튼

---

### Template F: 정보/약관 페이지
**사용 페이지:**
- `/guide/guide1` (이용약관)
- `/guide/guide2` (쇼핑몰 이용약관)

**특징:**
- 텍스트 중심 레이아웃
- 조항별 번호 체계 (제1조, 제2조...)
- 최소한의 스타일링
- 관련 약관 링크

---

### Template G: 인증 페이지
**사용 페이지:**
- `/login`

**특징:**

- 아이디/비밀번호 찾기 링크
